Instructions

  1. Select an abstract from arxiv, and using the first and second identifiers from the url, enter into the two form fields eg., if the url is https://arxiv.org/abs/xxxxx-xx/yymmnnn, then the two fields are xxxxx-xx and yymmnnn
  2. If extracted figures are found for the particular document, then these are returned as a list of available spreadsheets, and each can be examined for download by selection.
  3. When selected, a QC image is presented in the right column for the particular figure in question. The image shows the results of fitting a Gaussian mixture model to the pixel positions in the binarized image.
